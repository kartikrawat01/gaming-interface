How To Play! 🌈

🎮 Game Rules:

  • Tap a jar, then tap another jar to pour water.
  • You can only pour if the topmost liquid is the exact same color or if the target jar is empty.
  • Avoid getting stuck! Use Undo or Restart if needed.
  • How to Win: Sort all colors perfectly so each jar contains only one single color!
